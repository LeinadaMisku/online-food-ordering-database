# Online Food Ordering Database

This project presents the design and implementation of a database for an online food ordering system using Oracle SQL.The database stores and manages information about users, restaurants, menus, products, orders, couriers, and discount codes. Its purpose is to organize the food ordering and delivery process efficiently.

## Database Tables
The system includes the following main tables:
- Perdoruesi (Users)
- KodUlje (Discount Codes)
- Porosi (Orders)
- Produkte (Products)
- Menu
- Restorant
- Dergues (Courier)
- Takim (Delivery)

## Features
- ER/EER database design
- Table creation using SQL
- Primary and Foreign Keys
- SQL queries
- Views and indexes

## Technologies Used
- Oracle SQL
- SQL Developer
- Database Design (ER/EER Modeling)
