# Online Food Ordering Database System

This project presents the design and implementation of a relational database for an online food ordering system.

## Project Description
The database is designed to manage:
- Customers
- Restaurants
- Orders
- Payments
- Menu items

It includes:
- Entity-Relationship (ER) Diagram
- Table structures
- Primary and foreign keys
- SQL queries

## Technologies Used
- MySQL
- SQL
- Database Design Principles
